⚙ Instalar Caddy en el VPS

Caddy es el reverse proxy que maneja SSL automático. Hay que instalarlo una sola vez:

📋 Comandos paso a paso

1. Conectate al VPS por SSH y crear el container Caddy (elegí UNO de los dos modos):

Opción A — Caddy con docker run (rápido):

mkdir -p /opt/kumavpn/caddy /opt/kumavpn/caddy-data /opt/kumavpn/caddy-config

docker run -d \
  --name caddy-monitormaat \
  --restart unless-stopped \
  --network kumavpn-admin_default \
  -p 80:80 -p 443:443 -p 443:443/udp \
  -v /opt/kumavpn/caddy/Caddyfile:/etc/caddy/Caddyfile \
  -v /opt/kumavpn/caddy-data:/data \
  -v /opt/kumavpn/caddy-config:/config \
  caddy:2-alpine

Opción B — Agregar al docker-compose existente:

Edita /opt/kumavpn-docker/docker-compose.yml y agregá un servicio caddy con el mismo network. Después docker compose up -d.

2. Volvé a esta pantalla y guardá los dominios.

El panel detecta el container Caddy y aplica el Caddyfile automáticamente.

3. Verificá:

docker logs -f caddy-monitormaat

Vas a ver los pedidos a Let's Encrypt y los certs emitidos.
