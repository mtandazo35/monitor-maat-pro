Bot

Te permite enviar notificaciones automáticas (creación de usuarios, eventos de tenants, fallos) a un chat de admin y a cada usuario individual. Datos guardados en la DB.

¿Cómo obtener el token y el chat_id?
  1. Abrí Telegram, hablale a @BotFather, mandá /newbot y seguí las instrucciones. Te da un token tipo 123456:ABC-DEF....
  2. Iniciá una charla con tu bot (/start).
  3. Para el chat_id del admin: hablale a @userinfobot o @get_id_bot — te devuelve tu ID numérico.
  4. Cargá los dos datos acá y mandá un test.
  5. Para que cada cliente reciba notificaciones, cada uno tiene que cargar su propio chat_id en Mi password → Telegram.
